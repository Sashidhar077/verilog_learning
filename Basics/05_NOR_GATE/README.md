# 2-Input NOR Gate Simulation with Loop-Based Testbench

A simple hardware description language (HDL) implementation of a 2-Input NOR Gate in Verilog, featuring an automated, loop-based testbench for efficient verification.

## 📌 Project Overview
This project demonstrates the foundational concepts of digital logic design and automated verification. Instead of manually applying stimuli, the testbench utilizes a procedural `for` loop to cycle through all possible binary input combinations.
